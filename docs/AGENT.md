# Agent SOP (Standard Operating Procedure)

This file is the entry point for any AI agent working on this repository.

## General Instructions

1. **Familiarization**: Before making any changes, read the [wiki](https://github.com/julesklord/arbol/wiki).
2. **Compliance**: Strictly follow the laws defined in the [Agent SOP](https://github.com/julesklord/arbol/wiki/Agent-SOP). No exceptions.

## Agent Initialization Commands

1. Working on the Go version:
   - Ensure you are working under the Go modules system (go 1.20+).
   - Use `make build` to verify compiling.
2. Run unit tests (`go test ./...`) and integration tests (`make test`) before proposing changes.
